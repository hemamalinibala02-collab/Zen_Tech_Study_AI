# ZenTech AI – AI Powered Study Buddy

A Streamlit app for students: planner, notes, OCR, AI tutor, quizzes, revision, analytics, prediction, export.

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

Optional: copy `.env.example` to `.env` and set `OPENAI_API_KEY`.
Without a key, the app runs in **demo mode** (mock AI answers).

For OCR, install Tesseract on your system (https://github.com/tesseract-ocr/tesseract).
